# WriterLK MERN Monorepo

This repository hosts a MERN (MongoDB, Express, React, Node.js) application for an online business showcase.

## Structure

- `server/` — Node/Express API server
- `client/` — React frontend app
- `shared/` — Shared code (constants, utils)

## Next Steps

- Initialize packages and install dependencies for `server` and `client`.
- Set environment variables in `server/.env` based on `server/.env.example`.
- Start the API and the web client locally.
